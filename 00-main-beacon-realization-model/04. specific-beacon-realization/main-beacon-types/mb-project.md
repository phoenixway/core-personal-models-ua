## D. Орієнтир-проект

Приклад:

- написати і опублікувати одну рок-пісню високого рівня
    

Це класичний проект:

- є завершений результат
    
- є етапи
    
- є критерій done

## 4. **Проектні орієнтири**

Конкретні великі результати.

Питання:

- що саме має бути створено/завершено?
    

Приклади:

- опублікувати пісню
    
- випустити продукт
    
- завершити дослідження

## D. Алгоритм для проектного орієнтиру

Приклад: написати і опублікувати пісню

### Вхід:

конкретний бажаний результат

### Перетворення:

1. Уточнити definition of done.
    
2. Розбити на етапи.
    
3. Визначити залежності.
    
4. Оцінити мінімальну життєздатну версію.
    
5. Спустити в roadmap і задачі.
    

### Вихід:

- roadmap
    
- milestone’и
    
- action units
