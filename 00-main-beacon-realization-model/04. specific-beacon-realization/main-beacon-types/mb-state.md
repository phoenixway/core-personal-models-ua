## B. Орієнтир-доменний результат

Приклад:

- слід життя: список доменів і бажаних результатів
    

Це вже конкретніша форма:

- у яких доменах
    
- що саме хочеш отримати
    
- який слід там має лишитися
    

Це не просто вісь, а **карта бажаних станів по сферах**.

## 2. **Станові орієнтири**

Бажаний тип життя або стану.

Питання:

- у якому стані має бути ця сфера?
    
- як має виглядати “добре” в цій області?
    

Приклади:

- здорові відносини
    
- фінансова стійкість
    
- сильна професійна форма
    

## B. Алгоритм для станового орієнтиру

Приклад: мати сильну професійну форму, добрий стан дружби

### Вхід:

опис бажаного стану

### Перетворення:

1. Описати, як виглядає “добре”.
    
2. Визначити індикатори стану.
    
3. Знайти, що цей стан підтримує, а що руйнує.
    
4. Виділити регулярні дії підтримки.
    
5. Задати правила контролю та корекції.
    

### Вихід:

- модель бажаного стану
    
- рутинні дії підтримки
    
- тригери корекції
